# console.log()란 무엇인가?
- console.log()는 괄호안의 메시지를 콘솔창에 출력하는 명령어다.
- 가장 대표적인 디버깅 방법으로, 자바스크립트가 출력을 하도록 도와주는 기능이다.
- 콘솔창 내에 숫자, 문자, true 등 여러 값을 인수로 전달한다.
```javascript
let a = 1;
let b = 'hello';
let c = true;

console.log(a);
console.log(b);
console.log(c);
console.log(a,b,c);
console.log(`${a}는 숫자 ${b}는 문자열`);
```
